# Geoelektrik & Elektromagnetik

Dieses Repository enthält das (deutschsprachige) Skript zum Modul "Geoelektrik & Elektromagnetik" für Studierende des Bachelorstudienganges "Geoinformatik und Geophysik" am Institut für Geophysik und Geoinformatik der TU Bergakademie Freiberg. Es dient als "Buch zur Vorlesung" als Vorbereitung auf die ´Prüfung, steht darüber hinaus natürlich allen Interessierten zur Verfügung. Eine englische Übersetzung ist angedacht.

Autoren: Thomas Günther & Ralph-Uwe Börner

Hier befinden sich die Quelldateien des mit dem Open-Source Publikationssystem [Quarto](https://quarto.org) erzeugten Skripts. Das Skript selbst (html-Output) befindet sich in der [Skript-Sammlung](https://tubaf-em.github.io/) der Github Gruppe [TUBAF-EM](https://github.com/TUBAF-EM) unter https://tubaf-em.github.io/Geoelektrik_Elektromagnetik/
